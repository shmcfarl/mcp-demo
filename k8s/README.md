# Kubernetes Resources

This directory contains Kubernetes manifests and related resources for the mcp-demo project.

## Directory Structure

- `manifests/` - Kubernetes manifest files (deployments, services, etc.)
- `helm/` - Helm charts (if applicable)
- `kustomize/` - Kustomize configurations (if applicable)

## Usage

This directory will be used to store and manage all Kubernetes-related configurations and resources for the project.
